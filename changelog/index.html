Save File Compatibility

Which Save Files Carry Over?

Kompas Productions changed the save format once between v20 and v23 — at the v22 major release. Here is what that means in practice.

  • v22.0.0 → v22.5.0 → v22.6.0 → v24.0.0: Save files carry over directly. No action needed.
  • v21.x → v22.0.0: Saves are not compatible. Start a new game or use a shared save file from the community.
  • v20.x → v21.x: Compatible within the v20–v21 range only.

Before any update, back up your save folder. On Android the default path is /Android/data/com.kompasproductions.summertimesaga/files/saves/. On PC it lives in your user documents folder under Summertime Saga/saves/.

On Mac, they live at ~/Library/RenPy/SummertimeSaga/saves/. Copy this entire folder to a safe location before updating.

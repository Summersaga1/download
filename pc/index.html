Windows Guide

How to Install Summertime Saga on Windows

Three steps. Under 5 minutes. Works on Windows 7 through Windows 11.

01

Download the Windows EXE

Go to the download page and select the Windows version. The file is an EXE installer approximately 1.5–1.8 GB in size. Download it directly to your Downloads folder or Desktop — anywhere easy to find. Use a stable connection; Wi-Fi is fine.

02

Handle the SmartScreen Warning

When you double-click the EXE, Windows SmartScreen may show a blue warning: "Windows protected your PC." This appears for any downloaded EXE that has not built up a reputation with Microsoft's servers — it is not a malware detection. Click "More info" then "Run anyway" to proceed with the install.

03

Run the Setup Wizard

Follow the on-screen prompts. Choose your install directory — the default location works fine. The installer extracts the game files and creates a desktop shortcut. Installation takes 1–3 minutes depending on your drive speed. No restart is required.

Antivirus flags: Some Windows antivirus tools flag the EXE as a false positive due to the adult content classification or because it is a newly downloaded executable. If your antivirus quarantines the file, add an exclusion for the game's install folder or temporarily disable real-time protection during installation. The file does not contain malware.

Save file location on Windows: Saves are stored at %APPDATA%\RenPy\SummertimeSaga. Paste this path into the Windows Explorer address bar to find it. Back up this folder before updating to a new version.

Mac Guide

How to Install Summertime Saga on Mac

Works on Intel Macs and Apple Silicon (M1, M2, M3). macOS 10.13 or higher required.

01

Download the Mac ZIP

From the download page, select the Mac version. The file downloads as a ZIP archive. Safari will auto-extract it to your Downloads folder. If you are using Chrome or Firefox, right-click the ZIP and select "Open With → Archive Utility" to extract it manually.

02

Move to Applications

Inside the extracted folder you will find SummertimeSaga.app. Drag this file into your /Applications folder. This is the standard Mac app installation method — no installer wizard needed. You can delete the ZIP and extracted folder after moving the app.

03

Bypass Gatekeeper on First Launch

macOS Gatekeeper blocks apps from unidentified developers by default. When you first open the app, you will see "SummertimeSaga cannot be opened because it is from an unidentified developer." To bypass: go to System Settings → Privacy & Security, scroll down, and click "Open Anyway" next to the blocked app entry. You only need to do this once.

Apple Silicon (M1 / M2 / M3) users: If the app does not open after Gatekeeper approval, right-click the app in Finder and select "Open" from the context menu instead of double-clicking. This forces the Gatekeeper override prompt to appear. On some macOS versions the "Open Anyway" button in Settings only appears after attempting to open the file first.

Save file location on Mac: Saves are stored at ~/Library/RenPy/SummertimeSaga/. Open Finder, press Cmd + Shift + G, and paste that path to navigate there. Back up this folder before installing a new version.

Troubleshooting

Common PC Installation Problems — Fixed

The issues that come up most often on Windows and Mac, and how to resolve each one.

Windows SmartScreen Blocks the EXE

Click "More info" on the SmartScreen dialog, then "Run anyway." SmartScreen flags newly downloaded EXE files that have not built up reputation data with Microsoft. This is not a malware detection — it is a reputation-based filter. The warning is expected and safe to bypass here.

Antivirus Quarantines the File on Windows

Add the game's install folder to your antivirus exclusion list before or after installation. In Windows Defender: go to Windows Security → Virus & Threat Protection → Manage Settings → Exclusions → Add an Exclusion → Folder, then select the game directory. Restore the quarantined file from your antivirus history if needed.

"Cannot Be Opened" on Mac — Gatekeeper

Go to System Settings → Privacy & Security and click "Open Anyway" next to the blocked entry. If you do not see the button, try opening the app first (it will fail), then check Settings — the button appears after a failed open attempt. Alternatively, right-click the app in Finder and select "Open" to force the override prompt.

Game Launches But Crashes Immediately

This is almost always a storage issue — the game needs roughly 2 GB free to operate. Check your available disk space and clear room if needed. If storage is fine, try running the EXE as Administrator on Windows (right-click → Run as administrator). On Mac, check that the app has permission to access your Documents and Downloads folder in System Settings → Privacy & Security → Files and Folders.

Black Screen or Display Issues on Launch

Try launching the game in windowed mode. On Windows, hold Shift while clicking the game shortcut to access launch options, or look for a config file in the game folder to set windowed mode. Update your GPU drivers if the black screen persists. On older integrated graphics, the game may default to a fallback renderer — this is normal and does not affect gameplay.

Save Files Missing After Update

Check the save folder paths listed in the Save Files section above. On Windows: %APPDATA%\RenPy\SummertimeSaga\. On Mac: ~/Library/RenPy/SummertimeSaga/. Saves are almost never deleted by an update — they are usually still in the folder. If the game is not detecting them, confirm the new version is installed in the same directory as the old one.
